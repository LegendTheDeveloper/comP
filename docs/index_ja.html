あなたのAIエージェントに、
完璧なローカルコード文脈を。

comP（コンプ）は、AIエージェントがコードの構造、依存関係、全文情報を完全に把握できるように支援する、VSCode拡張機能およびRust製バックエンドデーモンです。

使ってみる GitHubで見る

主な機能とテクノロジー

100%ローカルで動作し、エージェントへ提供するトークンサイズを最大80%節約します。

🔍

インテリジェントBM25検索

コードやマークダウン、Office文書に加え、PDFやデータ分析用のParquet (.parquet) ファイルも正確にトークン化し、高精度な全文キーワード検索を可能にします。

⚡

Rust製超高速デーモン

裏で動作する軽量バックエンドはRust製。数千ファイル規模のリポジトリもSQLite上に高速にグラフデータベース化し、エージェントの問合せに即時に応答します。

🛡️

堅牢なエラーハンドリング

ドキュメントファイルの一部が破損していても、デーモンはエラーをスキップしてスキャンを継続します。クラッシュフリーで信頼性の高いインデックス作成を実現します。

💼

多様なドキュメント・データ対応

Office（Word, Excel, PowerPoint）やPDFファイル、Parquetファイルのスキーマ情報を自動抽出・インデックス化。コード以外のデータ・文脈も網羅してエージェントに提供します。

VSCodeに完全統合

VSCodeサイドバーからボタンひとつで簡単に開始でき、コードの変更を監視してリアルタイムにインデックスを差分更新（stale状態の自動付与）します。

Claude Code, Cursor, Cline, Windsurf, GitHub Copilotなど、あらゆる主要AIエージェントのMCP（Model Context Protocol）設定を自動生成し、シームレスに連携可能です。

ASTコード圧縮 & @comp チャット連携

長大なAPI定義やクラスファイルをチャットに流し込みたい時、コマンド一つでコードをAST圧縮（コメント・空行の除去、関数本体のスケルトン化）してクリップボードにコピーできます。

さらに、VSCode Chat Participant APIを利用した @comp 連携により、添付ファイルを自動でAST圧縮しながらLLMにプロンプトとして送信可能。モデルの入力トークン制限を賢く回避し、消費コストを最大80%節約します。

クイックスタート (Quick Start)

数ステップであなたのVSCode環境にAIコンテキストインデクサーを導入できます。

1

拡張機能のインストール

リリースされた .vsix ファイルをダウンロードし、VSCodeにドラッグ＆ドロップするか、次のコマンドでインストールします：

code --install-extension comP-0.5.0.vsix
2

プロジェクトでデーモンを起動

プロジェクトフォルダを開き、VSCodeサイドバーの「Start comP」ボタンを押すことで、バックエンドのインデクサーが起動し、自動的にインデックスが開始されます。

3

エージェントを紐付ける

コマンドパレット → comP: Setup Agents を実行し、エージェントを選択。生成されたファイルを以下の手順で有効化：

  • Claude Code: claude mcp add comp <daemon-path> -e COMP_WORKSPACE_ROOT=<workspace> を実行
  • Cursor: ~/.cursor/mcp.json または .cursor/mcp.json にマージ
  • Cline: VSCode設定 → Cline › MCP Servers に貼り付け
  • Windsurf: ~/.codeium/windsurf/mcp_config.json にマージ
  • GitHub Copilot: .vscode/mcp.json に自動書き込み済み
  • Antigravity: ~/.gemini/antigravity-ide/mcp_config.json に自動書き込み済み
