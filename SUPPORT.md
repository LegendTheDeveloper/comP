# Support Policy

## Commercial status

comP is **free, open-source software** (MIT license). It is developed and maintained
on a voluntary basis with no commercial activity, no paid support tier, and no
mandatory fees. This places comP outside the scope of the EU Cyber Resilience Act
under Regulation (EU) 2024/2847 Art. 2(5)(a).

## Support channels

| Channel | Purpose |
|---|---|
| [GitHub Issues](https://github.com/tsucky230/comP/issues) | Bug reports, feature requests |
| [GitHub Discussions](https://github.com/tsucky230/comP/discussions) | Questions, usage help |
| [SECURITY.md](SECURITY.md) | Vulnerability reports (private) |

There is no SLA. Responses are best-effort.

## Supported versions

The latest release on the `main` branch is the only actively maintained version.
Older versions receive no backported fixes.

## End-of-life

There is no scheduled end-of-life date. If the project is discontinued, a notice
will be posted in the GitHub repository at least 30 days in advance.
